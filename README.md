## 🧑🏻‍💻 Tecnologias

Projeto desenvolvido com as seguintes tecnologias:

- HTML e CSS
- JavaScript
- Git e Github

## 💻 Projeto

Inclui uma interface simples, onde o usuário seleciona a moeda de origem, insere o valor, e o aplicativo converte automaticamente para Reais.
